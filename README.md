# Asset Manager System 📦

A simple, interactive console-based **Asset Manager** built using Python and Object-Oriented Programming (OOP) concepts. This project tracks inventory levels and provides automated low-stock alerts.

### Features
- **Add Assets:** Store items with unique IDs, names, and custom stock thresholds.
- **Update Inventory:** Modify quantities with built-in validation to prevent negative stock values.
- **Low Stock Alerts:** Get notified automatically when an item drops below your set threshold.


### How to Use
1. Run the script: `python asset_manager.py`
2. Select an option from the menu (Add, Update, Display, or Exit).
3. Follow the on-screen prompts to manage your inventory.

### OOP Concepts Used
- **Classes & Objects:** Data encapsulation for each asset.
- **Methods:** Modular logic for updates and alerts.
- **Data Integrity:** Validation checks to ensure reliable data handling.
